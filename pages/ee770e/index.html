虚拟机固定ip地址

# 虚拟机固定 IP 地址

# 第一步

  1. 进入配置文件
vi /etc/sysconfig/network-scripts/ifcfg-ens33
1
  1. 下面这部分，如果有，则修改，没有则添加进去
IPADDR=192.168.78.132 //这是自己要改成的ip地址
GATEWAY=192.168.78.2 //网关
DNS1=192.168.78.2 //DNS
NETMASK=255.255.255.0 //子网掩码
HWADDR=00:0C:29:86:63:0e //mac地址
1
2
3
4
5

  1. 然后，把 BOOTPROTO="dhcp" 修改成 BOOTPROTO="static"

  2. 重启网卡

    • ifdown ens33
    • ifup ens33

如果修改之后网卡激活不成功，连接不到网络，请参考第二步

# 第二步

按顺序执行以下指令

chkconfig NetworkManager off

chkconfig network on

service NetworkManager stop

service network start
